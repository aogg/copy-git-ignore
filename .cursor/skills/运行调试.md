# 运行调试技能

## 技能描述
此技能用于运行和调试copy-ignore工具，专门针对以下命令进行调试：
```
.\copy-ignore.exe --dry-run --exclude "*\node_modules" --exclude "*\.idea" --exclude "*\.history" --exclude "*\vendor" --exclude "*\runtime" --exclude "*\.gradle" --exclude "*\.vite" --exclude "*\assets" --exclude "*\build" --exclude "*\__pycache__" --exclude "*\logs" --exclude "*\.intellijPlatform" --exclude "*\*.log" --exclude "*\*.csv"  --exclude "*\nginx.htaccess" --exclude "*\*.phar" --verbose D:\code\www D:\code\www_ignore_backup
```
